# Student Academic Assistant

A mobile application designed for African Leadership University (ALU) students to streamline academic life management.

## Purpose

This app helps ALU students stay organized and on top of their academic responsibilities by providing a centralized platform to:

- **Track Assignments**: Manage coursework deadlines, priorities, and completion status
- **Monitor Schedule**: View weekly class sessions, mastery sessions, study groups, and PSL meetings
- **Record Attendance**: Track attendance for all academic sessions
- **Stay Informed**: See current academic week and upcoming commitments at a glance

## Key Features

- **Assignment Management**: Create, view, and mark assignments as complete with due date tracking and priority levels
- **Schedule Overview**: Visual weekly schedule with session details (time, location, type)
- **Attendance Tracking**: Mark yourself present or absent for each session
- **Smart Dashboard**: Quick overview of pending assignments, today's schedule, and attendance statistics

## Tech Stack

- **Framework**: Flutter (cross-platform mobile development)
- **Storage**: Local persistence using shared_preferences
- **Platforms**: Android, iOS, Web, Linux, macOS, Windows
